# supply-chain-data-analysis-sql
SQL project analyzing supply chain data: revenue, inventory, suppliers, and shipping insights.

#Project Overview
This project analyzes a supply chain dataset to understand revenue, inventory, supplier performance, and shipping efficiency using SQL. It demonstrates practical data analytics skills relevant for a Data Analyst role.

## SQL Analysis
The SQL scripts performs analysis including:
- Calculating key metrics: total revenue, units sold, stock levels, and supplier performance
- Identifying high-demand, low-stock products
- Ranking products by revenue and calculating cost per unit
- Analyzing shipping times, costs, and transportation modes
- Aggregations, averages, and basic ranking to extract actionable insights

## Skills Demonstrated
- SQL
- MySQL Workbench
- Aggregations, Ranking, and Grouping
- Business Analysis
- Supply Chain & Inventory Analytics
